# Imágenes

Coloca aquí la imagen de fondo para la página de arquitectura.

**Nombre del archivo:** `arquitectura-hero.jpg` (o `.jpeg`, `.png`, `.webp`)

**Formato recomendado:**
- JPG o WebP para mejor compresión
- Resolución: 1920x1080 o superior
- Tamaño optimizado: menos de 500KB

Una vez que subas la imagen con el nombre `arquitectura-hero.jpg`, se usará automáticamente en la página.
