# Instrucciones para subir la imagen

**IMPORTANTE:** El archivo `arquitectura-hero.jpg` actualmente está vacío (0 bytes).

## Pasos para subir la imagen correctamente:

1. **Asegúrate de tener la imagen** en tu computadora
2. **Copia la imagen** a esta carpeta: `public/images/`
3. **Nombra el archivo:** `arquitectura-hero.jpg` (o `.jpeg`, `.png`, `.webp`)
4. **Verifica que el archivo tenga un tamaño mayor a 0 bytes**

## Formatos aceptados:
- `.jpg` / `.jpeg`
- `.png`
- `.webp`

## Tamaño recomendado:
- Resolución: 1920x1080 o superior
- Peso: menos de 2MB para mejor rendimiento

Una vez que subas la imagen correctamente, se mostrará automáticamente en la página de arquitectura.

**Nota:** Si el archivo tiene 0 bytes, la imagen no se mostrará y se usará la imagen de respaldo.
